# 歯科技工指示書 PWA

歯科技工指示書のHTMLフォームをPWA（プログレッシブ・ウェブ・アプリ）化したものです。

## 含まれるファイル

| ファイル | 役割 |
|---|---|
| `index.html` | 本体（PWA用メタタグ＋SW登録スクリプト追加済み） |
| `manifest.json` | アプリ情報（名前・アイコン・テーマカラー等） |
| `sw.js` | Service Worker（オフラインキャッシュ） |
| `icon.svg` | ベクターアイコン |
| `icon-192.png` / `icon-512.png` | PWA標準アイコン |
| `apple-touch-icon.png` | iOSホーム画面用アイコン（180px） |

## 動作要件

PWAは **HTTPS** または **localhost** でしか動作しません（Service Workerの仕様）。
ファイルを直接ダブルクリックで開いた `file://` 経由では動きません。

## 使い方

### 1. ローカルで試す
ファイル一式を置いたフォルダで簡易サーバーを起動：

```bash
# Python3 の場合
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開く。

### 2. 公開サーバーに置く
`index.html`、`manifest.json`、`sw.js`、各アイコン画像を**同じディレクトリ**に置けばOKです。
HTTPSで配信される場所（例：GitHub Pages、Netlify、Vercel、Cloudflare Pages、自前サーバー等）にアップロードしてください。

### 3. インストール（ホーム画面追加）

- **iOS Safari**: 共有ボタン →「ホーム画面に追加」
- **Android Chrome**: メニュー →「アプリをインストール」または「ホーム画面に追加」
- **デスクトップChrome/Edge**: アドレスバー右の「インストール」アイコン

インストール後はアプリアイコンから単独ウィンドウで起動できます。

## オフライン動作

初回アクセス時にHTML・アイコン・マニフェストがキャッシュされ、以後はオフラインでもフォームを開けます。本フォームは外部APIを呼ばない完結したUIなので、オフラインでも全機能が使えます。

## 更新方法

`index.html`等の中身を変更したら、`sw.js` の `CACHE_VERSION` を上げてください（例：`v1.0.0` → `v1.0.1`）。これで次回アクセス時に旧キャッシュが破棄され、新ファイルが読み込まれます。

## 注意事項

- 入力データはブラウザ内で完結しています。クラウド同期はありません。
- PWAインストール後にコードを差し替える際は、Service Workerが旧版をキャッシュしているため、上記「更新方法」のバージョン更新が必要です。
